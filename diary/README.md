# 📓 Project Diary - Life Assistant V4

Este directorio contiene el registro diario del desarrollo del proyecto.

## 📋 Estructura

- **Sesiones individuales:** `YYYY-MM-DD.md`
- **Resúmenes semanales:** `week-NN-YYYY.md`
- **Auditorías:** Enlazadas desde raíz del proyecto

## 🎯 Propósito

- Registrar progreso diario
- Documentar decisiones técnicas
- Trackear tiempo dedicado
- Listar tareas completadas
- Planificar próximas sesiones

## 📊 Métricas

Cada entrada debe incluir:
- ⏱️ Tiempo dedicado
- ✅ Tareas completadas
- 🐛 Bugs encontrados/resueltos
- 💡 Aprendizajes
- 🎯 Plan siguiente sesión
